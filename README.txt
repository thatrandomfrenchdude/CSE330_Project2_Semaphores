This program illustrates multi-threading using semaphores in two ways.

Method 1: A single semaphores controls two distinct operations.
Method 2: Two semaphores control a producer-consumer scenarioproj-2.c contains the main functions along with the functions, variable, and semaphore definitions.

sem.h contains the semaphore data structure an function definitions.

q.h contains the queue functions.

TCB.h contains the thread control block functions.

threads.h contains the thread functions.
